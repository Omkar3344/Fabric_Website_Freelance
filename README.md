# Fabric Loom Website

A light-mode marketing site for a premium fabric manufacturer, built with Astro and Tailwind CSS.

## Key Features
- Home, About, Products, and Contact pages with smooth in-page navigation.
- Fabric categories (sweaters, casuals, traditionals, embroidery, seasonal).
- Search component for products or global content.
- SEO-friendly metadata, semantic sections, image alt text, and Open Graph tags.
- Scroll-triggered reveals, parallax hero imagery, and Tailwind-powered theming.

## Color Palette

## Tech Stack
- Astro 4.x
- Tailwind CSS with custom theme extensions
- TypeScript-ready components
- ESLint & Prettier (if enabled)

## Development Scripts
| Command            | Description                |
|--------------------|----------------------------|
| `npm install`      | Install dependencies       |
